Real-time Weather Forecast API

You can try entering each route using the routes given below

  • Use "city_name" for location_id. For Example:- /weather/mumbai
  • GET /locations : Get all locations user has added along with some default locations.
  • POST /locations : User can also add location by providing location_id : "city_name" in body
  • GET /locations/:location_id : User can get weather forecast of that location provided in location_id.
  • PUT /locations/:location_id : User can update the location by changing the location_id to city he wants in body.
  • DELETE /locations/:location_id : User can delete the location.
  • GET /weather/:location_id : User can get weather forecast of any location provided in ID.
  • GET /history/:location_id : Get the historical data. Provide city name and country in query.
